Музыкальный стриминговый сервис на Python
Описание

Это мини-музыкальный сервис, реализованный на Python с использованием объектно-ориентированного подхода. Программа позволяет создавать пользователей, альбомы, треки и плейлисты, управлять ими и сохранять данные между запусками через JSON-файлы.

Проект полностью работает в консольном интерфейсе, с возможностью поиска треков по названию или исполнителю.
